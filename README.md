# Tome do Mestre - Projeto de Engenharia de Software

## Descrição

Este projeto é um sistema de gerenciamento de sessões de para mestres de RPG, desenvolvido para a disciplina de Engenharia de Software. A arquitetura foi projetada utilizando um modelo de microsserviços separados, focando em escalabilidade, tolerância a falhas e desacoplamento.

## Arquitetura Proposta

A solução utiliza uma arquitetura moderna e coesa, baseada nos seguintes serviços e tecnologias da AWS:

* **Middleware:** Express.js
* **Virtualização:** Docker
* **Comunicação:** REST APIs
* **Nomeação:** DNS de Contêineres
* **Banco de Dados:** PostgreSQL
* **Tolerância a Falhas:** Reinicialização de Contêineres
* **Segurança:** Health Checks

## Funcionalidades Planejadas

* [ ] Cadastro e Autenticação de Usuários
* [ ] Criação e gerenciamento de campanhas
* [ ] Envio de convites para jogadores
* [ ] Sistema de aceite/recusa de convites
* [ ] Listagem e visualização de sessões onde o usuário participa
